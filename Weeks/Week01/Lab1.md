# Lab 1: Let's make an interactive web map

Open VS Code

Go to File, New File (or ctrl-N/command-N)

Enter the following code:

```html
<!DOCTYPE html>
<html>
<head>
	<title>Hello World</title>
	<meta charset="utf-8" />

	<!-- style sheets -->
	<link rel="stylesheet" href="css/style.css">

</head>
<body>

	<div class="header">
		Hello World
	</div>
	<div class="sidebar">
		
	</div>
	<div class="content">

	</div>

</body>
</html>
```
